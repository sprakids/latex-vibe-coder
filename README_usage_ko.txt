LaTeX Vibe Coder 사용법

이번 버전은 LaTeX를 거치지 않고 PDF를 바로 생성합니다.
MiKTeX, TeX Live, xelatex 설치가 필요 없습니다.
출력 양식은 평가원 수능 영어 문제지에 가깝게 큰 시험지 크기, 2단 편집, 한컴바탕/Times 계열 폰트를 사용합니다.

1. LaTeXVibeCoder.exe를 실행합니다.
2. 왼쪽 문항 목록에서 문항을 추가, 복제, 삭제합니다.
3. 오른쪽 편집 영역에서 유형, 번호, 발문, 지문, 선택지, 배점을 수정합니다.
4. 가운데 문제지 배치 화면에서 문항 카드를 드래그하면 PDF 출력 순서와 열이 바뀝니다.
5. 파일 > PDF 바로 만들기 또는 하단의 PDF 바로 만들기 버튼을 눌러 PDF를 저장합니다.
6. 프로젝트 JSON 저장/불러오기를 쓰면 수업별 문제지를 이어서 편집할 수 있습니다.

지원 기능
- 평가원 수능 영어 스타일 2단 PDF 생성
- 원본 시험지와 같은 842 x 1191pt 페이지 크기
- 한글: 한컴바탕 계열, 영어: Times New Roman 계열 우선 사용
- 지문형, 듣기형, 안내문/도표, 표, 대화문, 자유형 문항 지원
- ①~⑤ 선택지 자동 출력
- [3점] 배점 표시
- 파이프(|)로 입력한 표를 PDF 표로 변환
- Apple Liquid Glass 느낌의 밝은 유리 패널 UI

예시 표 입력
| Club | Day | Time | Room |
| Debate | Monday | 4:00 | 201 |
| Film | Tuesday | 3:30 | 105 |
